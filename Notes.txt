LINQ----> 3 types of delegates
Action(void)-----upto 16 parameters ie 0 paramters or upto 16 parameters input
Func(any return type except bool)-----17 parameters ----last parameter is output parameter----upto 16 parameters input 
Predicate(only return type is bool,no other return  type)----1 parameter input & 1 parameter as output(bool)



	>=1 input parameter and u also want output from a method ---then its a func lambda
	only one input parameter and 1 output and that output is bool-----then its predicate lambda
